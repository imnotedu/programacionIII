# Imágenes de Productos

Esta carpeta contiene las imágenes de los productos de PowerFit.

## Estructura de nombres:
- `ON-WH-001.jpg` - Optimum Nutrition Gold Standard Whey Chocolate
- `DYM-ISO-002.jpg` - Dymatize ISO100 Vainilla
- etc.

## Formato:
- Formato: JPG o PNG
- Tamaño recomendado: 400x400px o 800x800px
- Peso máximo: 500KB por imagen

## Cómo agregar imágenes:
1. Guarda la imagen con el código del producto como nombre
2. Colócala en esta carpeta
3. Haz commit y push a GitHub
4. Las imágenes se desplegarán automáticamente en Render
